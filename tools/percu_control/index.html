FLASH FIRMWARE

Carga el firmware del PercuSynth directo al ESP32-S3 sin Arduino IDE, igual que WLED. Usa Web Serial + esptool.js.

✓ firmware.bin listo en firmware/

Para actualizar el firmware (cuando cambies el sketch):

  1. Abre el sketch en Arduino IDE
  2. Board: ESP32S3 Dev Module · Flash Mode: DIO · USB CDC On Boot: Enabled
  3. Sketch → Export Compiled Binary (Ctrl+Alt+S)
  4. Copia build/.../*.merged.bin a firmware/firmware.bin (sobrescribe)
⚠ ESP Web Tools no funciona con file://. Servir esta carpeta con un servidor local antes de instalar:
cd tools/percu_control && python -m http.server 8000
luego abrir http://localhost:8000
